# Design Directions

Directions are competing design proposals derived from briefs.

Each core surface should explore at least three materially different directions before one is accepted:

- content authority
- spatial object
- efficiency operation

Directions must include what they reject and why they belong to 软书四六级.

## Accepted Direction Sets

- `learning-surface-3-directions.md`: Learning surface A/B/C direction set and accepted `Knowledge Object` direction.
- `space-surface-visual-directions-v1.md`: Space surface A/B/C visual direction set and accepted `Box Desk` direction.
