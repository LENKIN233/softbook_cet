## Visual Evidence

## Product Truth

Which candidate better preserves the product truth? Name any blocker.

## Task Clarity

Which candidate makes the next user action clearer with lower burden?

## Space Or Interaction Fit

Which candidate better preserves Space hierarchy or the relevant interaction silhouette?

## Visual System Fit

Which candidate better preserves Law of One, focal object, state language, and forbidden-pattern avoidance?

## Implementation Mapping

Which candidate maps more cleanly to RN/Web without inventing design in code?

## Rationale

Explain the decision in concrete design terms.

## Borrowable Fragments

List fragments worth harvesting from either candidate.

## Rejected Fragments
